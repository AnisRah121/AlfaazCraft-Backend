Built with ❤️ to express emotions through words.

AlfaazCraft Backend is a Spring Boot REST API designed to power a modern poetry and shayari platform.

It provides endpoints for managing posts, handling user interactions, and integrating with a PostgreSQL database using Spring Data JPA.

Key Features:
- RESTful API architecture
- CRUD operations for posts
- PostgreSQL database integration
- JPA & Hibernate ORM
- Clean layered architecture (Controller, Service, Repository)

Tech Stack:
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
